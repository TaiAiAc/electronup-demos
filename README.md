# electronup-demos
electronup的示例demo

## 使用

使用 git 拉取模版到本地
```git
git clone https://github.com/TaiAiAc/electronup-demos
```

or

[跳转仓库](https://github.com/TaiAiAc/electronup-demos)下载zip
解压到本地

打开项目根目录下的命令行

如果没有 pnpm 先全局安装 pnpm
```sh
npm i -g pnpm
```

安装完成后 安装依赖
```sh
pnpm i
```

创建 demo
```sh
pnpm demo
```

输入想要创建的 demo 名称

创建完成后 
```sh
cd demos/[you-demo]
```

启动 demo
```sh
pnpm dev
```
